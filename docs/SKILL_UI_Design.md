# SKILL: 影片對話 HUD 工具 UI 設計

## 概述

這個工具採用 CustomTkinter 框架開發，提供一個現代化的暗色主題 UI，用於進行影片字幕對話的校稿和編輯工作。

## 主視窗架構

### 窗口配置
- **標題**: 「影片對話 HUD」
- **預設尺寸**: 1500x860 像素
- **最小尺寸**: 1180x720 像素
- **主題**: 暗色模式 (Dark)
- **配色**: 藍色主題
- **背景色**: `#060B14` (深藍黑)

### 佈局結構 (3 列布局)

```
┌─────────────┬────────────────────────┬──────────────┐
│   左側      │      中央預覽區        │    右側      │
│   邊欄      │      (canvas)          │  腳本面板    │
│   (220px)   │      (weight=1)        │  (可變寬度)  │
└─────────────┴────────────────────────┴──────────────┘
```

## 左側邊欄 (SideBar)

### 基本配置
- **寬度**: 固定 220px
- **背景色**: `#111827`
- **圓角**: 8px
- **邊距**: 外側 padx=(10, 6), pady=10

### 主要組件

#### 標題區 (Row 0-1)
- **標題**: 「影片對話 HUD」(20px 加粗)
- **副標題**: 「校稿工作台」(灰色 `#AAB0C0`)

#### 工作流程按鈕 (Row 2-11)
1. **btn_video** - 選擇影片 (高度 34px)
2. **btn_draw_people** - 人物標框
3. **entry_people_count** - 人物數量設定
4. **btn_scan** - 掃描對話
5. **entry_id** - 人物 ID 輸入
6. **entry_speaker** - 發言人名稱
7. **entry_text** - 對話文本
8. **btn_split** - 斷句按鈕
9. **btn_add** - 新增對話
10. **btn_export_hud** - 匯出 HUD
11. **btn_save_data** - 保存腳本

#### 可展開區域 (Row 12)
- **垂直伸縮** (`grid_rowconfigure(12, weight=1)`)
- **日誌面板** - 可折疊的日誌顯示

### 按鈕樣式
- **高度**: 34-40px
- **響應色**: 懸停時變亮
- **狀態**: 可根據工作流程禁用/啟用

## 中央預覽區

### Canvas 預覽畫布
- **用途**: 顯示影片幀和標註信息
- **模式**: 支援多種交互模式
  - **Pan (平移)** - 拖曳移動視圖
  - **ROI (矩形選擇)** - 繪製感興趣區域
  - **Bubble 拖曳** - 移動對話框位置

### 縮放機制
- **初始縮放**: `preview_zoom = 1.0`
- **偏移追蹤**: `canvas_offset = [0, 0]`
- **滑鼠拖曳**: 記錄起始位置和移動距離

### 控制信息
- **滑鼠位置顯示**: 實時座標
- **幀編號顯示**: 當前影片幀編號
- **時間戳顯示**: 對應的影片時間

## 右側腳本面板 (ScriptPanel)

### 面板配置
- **滾動容器**: CTkScrollableFrame
- **背景色**: 深色主題配合
- **邊距**: 主要為 padx=12, pady 根據內容調整

### 腳本行設計

#### 行容器
- **背景色**: 根據狀態變化
  - 正常發言者: 根據調色盤 (8 色輪轉)
  - 「無講話」: `#252A34` (灰色)
  - 被選中: `#28364A` (亮藍)
  - 被刪除: `#4A2630` (暗紅)

#### 行邊框
- **選中狀態**: 黃金色邊框 (`#FBBF24`)，寬度 1px
- **未選中**: 無邊框

#### 時間按鈕
- **寬度**: 98px
- **高度**: 28px (靜音行) 或 26px
- **背景**: `#334155` 正常，`#5B2B34` 刪除狀態
- **功能**: 點擊跳轉到該時間點

#### 發言人欄位
- **組件**: CTkComboBox (下拉式選單)
- **功能**: 可選擇現有人物或直接輸入新名稱
- **刪除狀態**: 顯示刪除線文本

#### 對話文本
- **展開式**: 大多數行顯示完整文本
- **靜音行**: 壓縮顯示，僅顯示時間 + 圖示

#### 操作按鈕
- **刪除/還原按鈕**: `btn_delete` (紅色) 或 `btn_restore` (藍色)
- **新增按鈕** (行尾): `btn_add_after` (綠色)

### 操作按鈕區

位於腳本面板頂部：
- **批量刪除**: 刪除所有「無講話」行
- **全選/反選**: 快速選擇對話
- **合併**: 合併相鄰行

## 時間軸區域

### 時間軸滑桿
- **位置**: 預覽區下方
- **類型**: CTkSlider
- **顯示值**: 當前幀編號
- **交互**: 拖曳跳轉，雙擊重置

### 波形顯示
- **上方區域**: 繪製音頻波形
- **功能**: 視覺化音頻活動
- **拖曳句柄**: 調整對話時間範圍

### 播放控制
- **播放按鈕**: 播放/暫停當前對話
- **音頻預覽**: 快速聆聽對話內容
- **音頻刮擦器**: 啟用/禁用音頻預覽

## 配色系統

8 組配色，根據說話者名稱雜湊輪轉：

1. 深青/淺青: `#173B45` / `#38BDF8`
2. 深紫/淺紫: `#3B2A50` / `#A78BFA`
3. 深綠/淺綠: `#23462F` / `#4ADE80`
4. 深橙/淺橙: `#4A341C` / `#F59E0B`
5. 深粉/淺粉: `#4A2638` / `#F472B6`
6. 深青綠/淺青綠: `#1F3F3A` / `#2DD4BF`
7. 深紅/淺紅: `#3E2D22` / `#FB7185`
8. 深藍/淺藍: `#29364D` / `#60A5FA`

### 無講話顏色
- **背景**: `#252A34` (中性灰)
- **強調**: `#64748B` (淡灰)

## 字體設定

### 主要字體
- **字族**: Microsoft JhengHei UI (微軟正黑體)
- **大小**: 12-20px
- **樣式**: 正常、加粗、刪除線

### 特殊樣式
- **刪除線**: 用於已刪除的對話項目

## 鍵盤快捷鍵綁定

| 快捷鍵 | 功能 |
|--------|------|
| Space | 播放/暫停 |
| ← / → | 逐幀前進/後退 |
| ↑ / ↓ | 上/下一句對話 |
| Ctrl+Z | 撤銷 |
| Ctrl+Y | 重做 |
| Home | 回到開始 |
| End | 跳到結尾 |
| Return | 播放當前句 |
| Delete | 刪除/還原當前句 |
| [ / ] | 微調句子邊界時間 |

## 回應式設計

### 斷點調整
- **寬度 < 1180px**: 警告最小尺寸
- **高度 < 720px**: 警告最小尺寸

### 彈性佈局
- 左側邊欄: 固定寬度
- 中央預覽: 主要伸縮區域 (weight=1)
- 右側腳本: 可變寬度但有最小值

### 元素縮放
- 按鈕: 高度和寬度根據內容自動調整
- 文本: 自動換行和截斷
- 圖片: 等比例縮放保持宽高比
